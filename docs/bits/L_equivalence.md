# equivalence

An equivalence relation is a relation that is [reflexive](https://github.com/conjure-cp/conjure/blob/main/docs/bits/L_reflexive.md), [symmetric](https://github.com/conjure-cp/conjure/blob/main/docs/bits/L_symmetric.md), and [transitive](https://github.com/conjure-cp/conjure/blob/main/docs/bits/L_transitive.md).

See this demonstrated [here](https://github.com/conjure-cp/conjure/blob/main/docs/notebooks/RelationDomains.ipynb).
Read more about this property [here](https://en.wikipedia.org/wiki/Equivalence_relation).
